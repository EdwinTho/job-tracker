## Codebase Patterns
- Use ES modules (`type="module"`) for JavaScript - all scripts use export/import
- CSS variables defined in :root for theming, with [data-theme="dark"] override
- Responsive breakpoints: mobile <768px, tablet 768-1024px, desktop >1024px
- Minimum touch target size: 44px for buttons/interactive elements
- Use BEM-style naming for CSS classes (e.g., .header__title, .btn--primary)
- Storage key is 'jobTrackerData' - use Storage module from scripts/storage.js for all data operations
- Application statuses: 'Applied', 'Screening', 'Interview', 'Offer', 'Rejected'
- Cards use data-id attribute for referencing application ID - use escapeHtml() for user content
- Modal module: use Modal.openAddApplication() for add form, Modal.close() to close - call Kanban.render() after data changes
