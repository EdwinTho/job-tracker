# Job Application Tracker

A comprehensive web application to track job applications, interviews, and follow-ups with a Kanban board interface.

## Features
- Track applications through the hiring pipeline
- Kanban board view (Applied → Screening → Interview → Offer → Rejected)
- Company research notes and contact information
- Interview scheduling and reminders
- Application statistics and insights

## Tech Stack
- HTML5, CSS3, JavaScript
- Local storage for data persistence
- Responsive design for mobile and desktop

## Getting Started
Open `index.html` in your browser

## Built With
AI-assisted development using Ralph and Amp
